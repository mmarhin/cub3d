# textures/

Place your `.xpm` texture files here. Four files are required by the subject:

| File         | Subject identifier | Wall side |
|--------------|--------------------|-----------|
| north.xpm    | NO                 | North     |
| south.xpm    | SO                 | South     |
| west.xpm     | WE                 | West      |
| east.xpm     | EA                 | East      |

XPM files can be created with GIMP or any image editor that supports XPM export.
Make sure textures are square (e.g. 64×64 or 128×128) for best results.
